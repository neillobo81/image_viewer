# Phase 1 Design Questions

Which part(s) of the program are responsible for reading the contents of the
starting directory?

What happens when a new tag is created?

Which class is keeping track of the association between tags and files?

Which class keeps the list of tags that we can choose from?

Which class/classes is responsible for displaying the stuff to the screen?

How do we keep the tag information after the program closes?

In which class does the execution of the program start?

How do we store a log of all renaming ever done?

How do we change the directory of a photo?

How do we revert back to the previous set of tags?
